# iRime-bopomo-config

## 放置 iRime 所使用的方案 及 反查時引用的方案

### allbpm 純注音符號 

### bopomo 注音 

### bopomo_onion 注音洋蔥版

### unicode 統一碼 

### xiapin 蝦拼 

### cangie5 倉頡5代

### luna_pinyin 朙月拼音  不需要打聲調的拼音

### terra_pinyin 地球拼音  ( 需要打聲調的拼音 , rime內建的注音方案均依賴此方案)

### easy_en 英語詞庫  (一般用於中英文混合輸入)

### pinyin_simp 簡字拼音 
### stroke 五筆劃
